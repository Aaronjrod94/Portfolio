2021 Cumberland Womens Volleyball Season

This project is an overview of the 2021 Cumberland womenâ€™s volleyball season. They finished the season with a record of 29-10. To get an in-depth view on how the team performed overall for the games these dashboards were created to get an easy look at the numbers. The data was taken from the NAIA website, and it consists of box scores for each game that was played from August-November. The first visualization quickly shows the result of each game and how the team performed. The second dashboard reveals the stats of each player for the season and for each game they participated in. A calculation is also shown that reveals how valuable that player was in that game. These breakdowns can be used for players and coaches that want to easily understand how the athletes are performing and it can also be used to discover what areas of their game need to be worked on.

      Team Overview

        Player Reports

        This dashboard shows every player's stats as well as a player rating that shows the impact of each player while on the court. The rating is calculated by this formula (Assists + Digs + Kills + Points + Service Aces + Blocks - Errors - Ball Handling Errors - Block Errors - Reception Errors - Service Errors)/Sets Played)

          Project Background

          After the data for each game was extracted, it was then cleaned and formatted properly in excel with the use of macros. Additional data sets were used to obtain the total stats for Cumberland and their opponents. These box scores were queried and combined by using union all. A temp table was then created for additional stats that didn't appear in the other datasets and were then joined with a dataset that contained the total stats of each player for the entire season.
