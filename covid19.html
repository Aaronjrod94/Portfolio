Covid-19 Statistics

This project covers the numbers for the total number of cases, deaths, and the infection rate per country. The data was gathered from Our World in Data.

    Total Numbers

    From the data we can see that Europe has the highest number of deaths sitting at 1,598,454. That's 20% more than the continent of Asia which has a total of 1,278,334

      Infection Rate Per country

      In this map we can quickly see the infection rate for every country. In the United States, 21% of the population has been infected. In France that number is sitting at 25%

        Global Numbers Calculated in SQL

        This script shows how the data was queried and is calculated to find the global numbers that is used for this project. A full script of the SQL code can be found here. 

        
Select *
From PortfolioProject..CovidDeaths
Where continent is not null
order by 3,4
                
Select Location, date, total_cases, new_cases, total_deaths, population
From PortfolioProject..CovidDeaths
order by 1,2
                
--Total Cases VS vs Total Deaths
-- Shows likelihood of death if infected
                
Select Location, date, total_cases, total_deaths, (total_deaths/total_cases)*100 as DeathPercentage
From PortfolioProject..CovidDeaths
Where location like '%states%'
 order by 1,2
                
-- Total Cases vs Population
-- Shows percentage of population infected
    Select Location, date, total_cases, population, (total_cases/population)*100 as InfectionRate
    From PortfolioProject..CovidDeaths
    --Where location like '%states%'
    order by 1,2
                
-- Looking at Countries with Highest Infection Rate Compared to Population
Select Location, population, MAX(total_cases) as HighestInfectionCount, MAX((total_cases/population))*100 as InfectionRate
From PortfolioProject..CovidDeaths
group by Location, population
order by population desc
                
-- Countries With Highest Death Count per population
Select Location, MAX(cast(Total_deaths as int)) as TotalDeaths
From PortfolioProject..CovidDeaths
Where continent is not null
group by Location
order by TotalDeaths desc
